# Compliance Certifications

## Overview

DotEnv is working towards achieving industry-standard compliance certifications to meet enterprise security and regulatory requirements.

Key controls:

### ISO 27001

**Status**: Planned

- **Scope**: Information Security Management System (ISMS)
- **Timeline**: Target Q4 2024
- **Benefits**:
  - International recognition
  - Systematic security approach
  - Continuous improvement framework

### HIPAA Compliance

**Status**: Planned

- **Scope**: Healthcare data protection
- **Components**:
  - Administrative safeguards
  - Physical safeguards
  - Technical safeguards
  - BAA availability

### PCI DSS

**Status**: Under Evaluation

- **Level**: Service Provider Level 1
- **Scope**: Payment card data protection
- **Requirements**:
  - Network security
  - Access control
  - Regular testing
  - Security policies
