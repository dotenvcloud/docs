# Product Roadmap

DotEnv's product development roadmap and upcoming features.

## Vision

DotEnv aims to be the most developer-friendly, secure, and scalable secrets management platform, enabling teams to manage sensitive configuration with confidence across all environments and platforms.

## Current Focus

### Q1 2024

#### ✅ Completed

- PHP SDK release
- JavaScript/TypeScript SDK release

- Basic Docker integration
- GitHub Actions integration

#### 🚧 In Progress

## Feature Request Process

### How to Submit Requests

1. **GitHub Issues**: Open feature requests in our public repository
2. **Community Forum**: Discuss ideas with other users
3. **Enterprise Support**: Direct channel for enterprise customers
4. **User Feedback**: In-product feedback widget

### Evaluation Criteria

We evaluate feature requests based on:
- User impact and demand
- Security implications
- Technical feasibility
- Strategic alignment
- Resource requirements

### Community Involvement

- **Beta Program**: Early access to new features
- **User Advisory Board**: Quarterly meetings with power users
- **Open Source Contributions**: Selected components open-sourced
- **Public Roadmap Reviews**: Monthly community calls
